```Rick and Morty (UIKit) ```


Rick&Morty es una serie de televisión estadounidense que narra las aventuras de Rick, un peculiar y alocado genio; y Morty, su nieto tímido y poco inteligente. 
Esta aplicación intenta mostrar todos los diferentes contenidos de este programa de televisión.
Podrás buscar entre cientos de personajes, episodios y descubrir todos los planetas que crean este universo.
